About BetterER

BetterER is a simple to use tool for creating ER-Diagrams.

With BetterER you are able to create the following things:

  • ER-Diagrams
  • SQL Files
  • Maybe more to follow

The tool was originally developed by Hendrik Roskamp. View the original repository for more information
